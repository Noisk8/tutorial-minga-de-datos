# tutorial-minga-de-datos

Tutorial interactivo en Next.js + Tailwind para documentar ontologías en Wikidata (persona víctima y auto judicial).

## Scripts
- `npm run dev` — entorno local
- `npm run build` — build de producción
- `npm run start` — servidor de producción
- `npm run lint` — lint (requiere configurar ESLint si se desea)

## Estructura
- `app/` — páginas (`/` home, `/victima`, `/auto`)
- `tailwind.config.js` — paleta y tema
- `app/globals.css` — estilos base y componentes temáticos
